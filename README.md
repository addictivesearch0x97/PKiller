# PKiller

**🚀 Automated PairIP License Removal Tool for Android APKs**

**PKiller** is a lightweight, automated utility designed to remove **PairIP license checks** from Android APK files.
No complicated setup — just run, patch, and go.


### 🔧 Features

* **Automatic PairIP license patching**
* **Fast & minimal interaction**
* **Works directly in Termux**
* **Ideal for reverse-engineering workflows**

### 📥 Installation (TermuxVoid Repository)
```
curl -sL https://github.com/termuxvoid/repo/raw/main/install.sh | bash
```

### Install PKiller
```
apt update && apt install pkiller -y
```

### 📚 Usage
```
pkiller <your_app.apk>
```

> Patched APK will be generated automatically.
>
